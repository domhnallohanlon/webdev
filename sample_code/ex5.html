Hexadecmial colours work by specifying the amount of red, green and blue you want to mix together. By using hexadecimal (base 16) rather than decimal (base 10) for encoding colours it is possible to create over 16 million colours using just 6 bits of data.

RGBa works in a similar way, but with two main differences. Firstly, the colours are represented with decmial numbers (0 - 255) so more bits of information are typically used. Additionally, a fourth channel - or alpha channel, is used to specify the opacity of the colour. This is specified as a number between 0 and 1
