Horizontal Navigation Bar

In this example, all the styles for the navigation bar a stored in a separate CSS file called "navStyle.css".

To keep everything in your site organised it is a good idea to group similar files together, i.e keep all the style sheets in a "css" folder, keep all your pictures in an "images" folder, and so on. This will become more useful as our sites grow in size.
